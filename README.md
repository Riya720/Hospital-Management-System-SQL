# 🏥 Hospital Management System (SQL Project)

## 📌 Project Overview
This project is a Hospital Management System built using SQL. It manages hospital operations such as patients, doctors, appointments, treatments, billing, and payments.

The system helps transform raw hospital data into meaningful insights for better decision-making and improved patient care.

---

## 🗂️ Features
- Patient Management
- Doctor Management
- Appointment Scheduling
- Treatment Records
- Billing and Payments
- Data Analysis using SQL queries

---

## 🛠️ Technologies Used
- SQL (MySQL / PostgreSQL / SQL Server)
- Database Design (ER Model)

---

## 🧱 Database Structure
- Patients Table
- Doctors Table
- Appointments Table
- Treatments Table
- Billing Table
- Payments Table

---

## 📊 Sample Queries
- Total number of patients
- Revenue generated
- Doctor-wise patient count
- Monthly appointments analysis

---

## ▶️ How to Run
1. Open your SQL tool (MySQL Workbench / SQL Server)
2. Run `schema.sql`
3. Run `data.sql`
4. Execute queries from `queries.sql`
